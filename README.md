# git-commands

## Initialise Git
```console
git init
```

## Rename branch
```console
git branch -m master main
```

## Update local files
```console
git pull
```

## Add files
```console
git add .
```

## Commit files
#### Unsigned commit
```console
git commit -m "Commit message"
```
#### Signed commit
```console
git commit -m "Commit message" -S
```

## Push files
```console
git push
```
